# VitePress HackMD 書本模式範本

這是一個適合「教學網誌 / HackMD 類筆記 / 書本目錄模式」的 VitePress 範本。

## 需要先安裝

1. Node.js LTS
2. Git
3. VSCode

## 第一次使用

```bash
npm install
npm run docs:dev
```

打開終端顯示的網址，通常是：

```text
http://localhost:5173
```

## 更新網站

改完 Markdown 後：

```bash
git add .
git commit -m "update article"
git push
```

GitHub Pages 會自動重新部署。
