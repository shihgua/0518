# 文章操作
## 新增文章
### 第一步：先新增 .md 檔案

在 VSCode 左側檔案欄，找到：

```text
docs
```

假設要在 `A` 這個章節資料夾底下新增文章。

先找到：

```text
docs/A
```

在 `A` 資料夾上按右鍵，選：

```text
New File
```

輸入檔名：

```text
A0.md
```

也就是新增：

```text
docs/A/A0.md
```

---

### 第二步：寫文章內容

打開：

```text
docs/A/A0.md
```

輸入：

```md
# 新的文章

這是新的文章內容。
```

存檔：

```text
Ctrl + S
```

---

### 第三步：把文章加入左側目錄

打開：

```text
docs/.vitepress/config.mts
```

找到 `sidebar`。

原本可能長這樣：

```ts
sidebar: [
  {
    text: '開始',
    collapsed: false,
    items: [
      { text: '前置作業', link: '/A/A1' },
      { text: '文章操作', link: '/A/A2' }
    ]
  }
]
```

在 `items` 裡新增一行：

```ts
{ text: '新的文章', link: '/A/A0' }
```

變成：

```ts
sidebar: [
  {
    text: '開始',
    collapsed: false,
    items: [
      { text: '前置作業', link: '/A/A1' },
      { text: '文章操作', link: '/A/A2' },
      { text: '新的文章', link: '/A/A0' }
    ]
  }
]
```

注意：

```text
新增的文章檔案是 docs/A/A0.md
所以 link 要寫 /A/A0
```


## 刪除文章

### 第一步：先刪除 .md 檔案

假設要刪除：

```text
docs/A/A0.md
```

在 VSCode 左側檔案欄找到：

```text
docs/A/A0.md
```

對檔案按右鍵，選：

```text
Delete
```

---

### 第二步：再刪除左側目錄連結

打開：

```text
docs/.vitepress/config.mts
```

找到這一行：

```ts
{ text: '新的文章', link: '/A/A0' }
```

把它刪掉。

原本：

```ts
items: [
  { text: '前置作業', link: '/A/A1' },
  { text: '文章操作', link: '/A/A2' },
  { text: '新的文章', link: '/A/A0' }
]
```

刪掉後：

```ts
items: [
  { text: '前置作業', link: '/A/A1' },
  { text: '文章操作', link: '/A/A2' }
]
```


## 逗號規則

修改 `sidebar` 時，最常出錯的是逗號。

規則很簡單：

```text
同一層有多個東西，中間要加逗號
```

正確：

```ts
items: [
  { text: '前置作業', link: '/A/A1' },
  { text: '文章操作', link: '/A/A2' },
  { text: '新增', link: '/A/A3' }
]
```

錯誤：

```ts
items: [
  { text: '前置作業', link: '/A/A1' }
  { text: '文章操作', link: '/A/A2' }
  { text: '新增', link: '/A/A3' }
]
```

少了逗號，網站會出錯。
