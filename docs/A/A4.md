# 章節操作
## 新增章節

### 第一步：先新增章節資料夾

章節通常會對應一個資料夾。

例如要新增一個章節叫：

```text
演算法
```

在 VSCode 左側檔案欄，找到：

```text
docs
```

對 `docs` 按右鍵，選：

```text
New Folder
```

新增資料夾：

```text
algorithm
```

也就是：

```text
docs/algorithm
```

資料夾名稱建議用英文，不要用空白。

---

### 第二步：在章節資料夾內新增文章

在：

```text
docs/algorithm
```

上按右鍵，選：

```text
New File
```

新增：

```text
bfs.md
```

再新增：

```text
dfs.md
```

最後會變成：

```text
docs/algorithm/bfs.md
docs/algorithm/dfs.md
```

---

### 第三步：寫文章內容

打開：

```text
docs/algorithm/bfs.md
```

輸入：

```md
# BFS

這是 BFS 教學。
```

打開：

```text
docs/algorithm/dfs.md
```

輸入：

```md
# DFS

這是 DFS 教學。
```

記得存檔：

```text
Ctrl + S
```

---

### 第四步：把章節加入左側目錄

打開：

```text
docs/.vitepress/config.mts
```

找到 `sidebar`。

原本可能是：

```ts
sidebar: [
  {
    text: '開始',
    collapsed: false,
    items: [
      { text: '前置作業', link: '/A/A1' },
      { text: '文章操作', link: '/A/A2' }
    ]
  }
]
```

在 `sidebar` 裡新增一整塊：

```ts
{
  text: '演算法',
  collapsed: false,
  items: [
    { text: 'BFS', link: '/algorithm/bfs' },
    { text: 'DFS', link: '/algorithm/dfs' }
  ]
}
```

完整會變成：

```ts
sidebar: [
  {
    text: '開始',
    collapsed: false,
    items: [
      { text: '前置作業', link: '/A/A1' },
      { text: '文章操作', link: '/A/A2' }
    ]
  },
  {
    text: '演算法',
    collapsed: false,
    items: [
      { text: 'BFS', link: '/algorithm/bfs' },
      { text: 'DFS', link: '/algorithm/dfs' }
    ]
  }
]
```

注意：

```text
新增章節時，是在 sidebar 裡新增一整塊
新增文章時，是在 items 裡新增一行
```

## 刪除章節

### 第一步：先刪除章節資料夾

假設要刪除整個章節：

```text
docs/algorithm
```

在 VSCode 左側檔案欄找到：

```text
docs/algorithm
```

對資料夾按右鍵，選：

```text
Delete
```

這會把裡面的文章一起刪掉。

---

### 第二步：再刪除左側目錄章節

打開：

```text
docs/.vitepress/config.mts
```

找到整個章節：

```ts
{
  text: '演算法',
  collapsed: false,
  items: [
    { text: 'BFS', link: '/algorithm/bfs' },
    { text: 'DFS', link: '/algorithm/dfs' }
  ]
}
```

把整塊刪掉。

## 逗號規則

修改 `sidebar` 時，最常出錯的是逗號。

規則很簡單：

```text
同一層有多個東西，中間要加逗號
```

正確：

```ts
sidebar: [
  {
    text: '開始',
    collapsed: false,
    items: [
      { text: '前置作業', link: '/A/A1' }
    ]
  },
  {
    text: '演算法',
    collapsed: false,
    items: [
      { text: 'BFS', link: '/algorithm/bfs' }
    ]
  }
]
```

錯誤：

```ts
sidebar: [
  {
    text: '開始',
    collapsed: false,
    items: [
      { text: '前置作業', link: '/A/A1' }
    ]
  }
  {
    text: '演算法',
    collapsed: false,
    items: [
      { text: 'BFS', link: '/algorithm/bfs' }
    ]
  }
]
```

兩個章節中間少了逗號。
